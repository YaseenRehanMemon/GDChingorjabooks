8.1 Functions as a Rule or Correspondence

A function is a rule that assigns each element from a set \(X\) (the domain) to exactly one element in a set \(Y\) (the co-domain). We can write this as \(f: X \to Y\), where \(y = f(x)\).

  • The variable \(x\) is the independent variable.
  • The variable \(y\) is the dependent variable, as its value depends on \(x\).

8.1.1 Domain, Co-domain and Range of a Function

When defining a function \(f: X \to Y\):

  • The set \(X\) is called the domain. It represents all possible input values for \(x\).
  • The set \(Y\) is called the co-domain. It contains all possible output values.
  • The range is the actual set of all output values \((f(x))\) produced by the function for the given domain. The range is always a subset of the co-domain.

Finding the Domain

For real-valued functions, the domain is restricted if:

  • There is a variable in the denominator (cannot be zero).
  • There is a variable under an even root (cannot be negative).

8.1.2 Types of Functions

Polynomial Functions

A function \(p(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0\) where \(a_n \neq 0\) and \(n\) is a non-negative integer is a polynomial function of degree \(n\).

  • Linear Function: Degree 1 (e.g., \(f(x) = ax+b\)).
  • Quadratic Function: Degree 2 (e.g., \(f(x) = ax^2+bx+c\)).

Square Root Functions

Functions of the form \(f(x) = \sqrt{x}\) are square root functions. Their domain requires the term under the square root to be non-negative.

8.1.3 One-to-one, Onto, and Bijective Functions

These classifications describe specific properties of how functions map elements between sets.

  • One-to-one (Injective): Each distinct element in the domain maps to a distinct element in the co-domain. No two different inputs give the same output.
  • Onto (Surjective): Every element in the co-domain is mapped to by at least one element in the domain. The range is equal to the co-domain.
  • Bijective: A function that is both one-to-one and onto. Bijective functions are important because they have inverse functions.

8.2 Sketching Graphs of Functions

The graph of a function \(y = f(x)\) is the set of all points \((x, y)\) that satisfy the function, providing a visual representation of its behavior.

8.2.1 Graphs of Linear Functions

A linear function \(y = ax+b\) produces a straight line. Key points for sketching include the x-intercept (where \(y=0\)) and y-intercept (where \(x=0\)).

8.2.2 Graphs of Non-Linear Functions (e.g., Quadratic)

A quadratic function \(y = ax^2+bx+c\) creates a parabola. If \(a > 0\), the parabola opens upward; if \(a < 0\), it opens downward.

8.2.3 Graphs of Square Root Functions

Functions like \(y = \sqrt{x}\) form curves. The graph will only exist for values of \(x\) where the expression under the root is non-negative.

8.2.4 Intersecting Graphs

The points where two graphs intersect represent the solutions where \(f(x) = g(x)\). These can be found graphically by plotting both functions on the same coordinate plane.

8.3 Inverse Functions

An inverse function, denoted as \(f^{-1}(x)\), "undoes" the action of the original function \(f(x)\). If \(f(a) = b\), then \(f^{-1}(b) = a\).

Conditions for an Inverse Function to Exist

An inverse function \(f^{-1}(x)\) exists if and only if the original function \(f(x)\) is bijective (both one-to-one and onto).

The domain of \(f^{-1}(x)\) is the range of \(f(x)\), and the range of \(f^{-1}(x)\) is the domain of \(f(x)\).

Finding the Inverse Function

  1. Replace \(f(x)\) with \(y\).
  2. Swap \(x\) and \(y\).
  3. Solve the new equation for \(y\) in terms of \(x\).
  4. Replace \(y\) with \(f^{-1}(x)\).

Graphical Representation of Inverse Functions

The graph of an inverse function \(f^{-1}(x)\) is a reflection of the graph of \(f(x)\) across the line \(y=x\).

Chapter Summary

  • A function maps each input to exactly one output.
  • The domain is the set of inputs, the co-domain contains all possible outputs, and the range is the set of actual outputs.
  • Functions can be one-to-one (unique inputs yield unique outputs), onto (all co-domain elements are outputs), or bijective (both).
  • Graphs visually represent functions, with specific shapes for linear (straight line), quadratic (parabola), and square root functions.
  • An inverse function \((f^{-1})\) exists for bijective functions and reverses the mapping of the original function. Its graph is a reflection across \(y=x\).
