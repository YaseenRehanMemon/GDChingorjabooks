2.1 Introduction to Matrices

A matrix is a rectangular array of numbers, symbols, or expressions arranged in rows and columns. Matrices are fundamental tools in mathematics, statistics, engineering, and various sciences for organizing data and solving systems of linear equations.

Matrix Notation and Order

A matrix \(A\) with \(m\) rows and \(n\) columns is denoted as \(A = [a_{ij}]_{m \times n}\), where \(a_{ij}\) is the element in the \(i\)-th row and \(j\)-th column.

The order or dimension of the matrix is \(m \times n\).

\[ A = \begin{pmatrix} a_{11} & a_{12} & \dots & a_{1n} \\ a_{21} & a_{22} & \dots & a_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ a_{m1} & a_{m2} & \dots & a_{mn} \end{pmatrix} \]

2.2 Types of Matrices

Matrices are classified based on their structure and elements.

TypeDescriptionExample
Row MatrixA matrix with only one row.\(\begin{pmatrix} 1 & 2 & 3 \end{pmatrix}\)
Column MatrixA matrix with only one column.\(\begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix}\)
Square MatrixNumber of rows equals number of columns (\(m=n\)).\(\begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix}\)
Rectangular MatrixNumber of rows does not equal number of columns (\(m \neq n\)).\(\begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{pmatrix}\)
Null/Zero MatrixAll elements are zero.\(\begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix}\)
Diagonal MatrixSquare matrix with non-zero elements only on the main diagonal.\(\begin{pmatrix} 1 & 0 \\ 0 & 4 \end{pmatrix}\)
Scalar MatrixDiagonal matrix where all diagonal elements are equal.\(\begin{pmatrix} 5 & 0 \\ 0 & 5 \end{pmatrix}\)
Identity Matrix (\(I_n\))Scalar matrix where all diagonal elements are 1.\(\begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}\)
Upper TriangularSquare matrix with all elements below the main diagonal as zero.\(\begin{pmatrix} 1 & 2 \\ 0 & 3 \end{pmatrix}\)
Lower TriangularSquare matrix with all elements above the main diagonal as zero.\(\begin{pmatrix} 1 & 0 \\ 2 & 3 \end{pmatrix}\)

Special Matrices

  • Transpose (\(A^t\)): Obtained by interchanging rows and columns of \(A\).
  • Symmetric Matrix: A square matrix where \(A^t = A\).
  • Skew-Symmetric Matrix: A square matrix where \(A^t = -A\). (Main diagonal elements are zero).
  • Idempotent Matrix: A square matrix where \(A^2 = A\).
  • Nilpotent Matrix: A square matrix where \(A^p = 0\) for some positive integer \(p\).
  • Involutory Matrix: A square matrix where \(A^2 = I\).
  • Periodic Matrix: A square matrix where \(A^{k+1} = A\) for some positive integer \(k \ge 1\).
  • Hermitian Matrix: A complex square matrix where \((\bar{A})^t = A\).
  • Skew-Hermitian Matrix: A complex square matrix where \((\bar{A})^t = -A\).

2.3 Algebra of Matrices

Matrices can be combined through various algebraic operations, provided their dimensions are compatible.

2.3.1 Scalar Multiplication

Multiplying a matrix \(A\) by a scalar \(k\) means multiplying every element of \(A\) by \(k\).

\[ kA = [ka_{ij}] \]

2.3.2 Addition and Subtraction

Matrices can be added or subtracted only if they have the same order. The operation is performed element-wise.

\[ A+B = [a_{ij}+b_{ij}] \] \[ A-B = [a_{ij}-b_{ij}] \]

2.3.3 Multiplication of Matrices

The product \(AB\) of two matrices \(A\) (order \(m \times p\)) and \(B\) (order \(p \times n\)) is defined only if the number of columns in \(A\) equals the number of rows in \(B\). The resulting matrix \(C\) will have the order \(m \times n\).

The element \(c_{ij}\) in \(C\) is the sum of the products of corresponding elements from the \(i\)-th row of \(A\) and the \(j\)-th column of \(B\).

\[ c_{ij} = \sum_{k=1}^p a_{ik}b_{kj} \]

Matrix multiplication is generally NOT commutative (\(AB \neq BA\)).

2.4 Determinants

A determinant is a unique scalar value associated with every square matrix. It provides crucial information about the matrix, such as its invertibility.

2.4.1 Minors and Cofactors

  • Minor (\(M_{ij}\)): The determinant of the sub-matrix formed by deleting the \(i\)-th row and \(j\)-th column.
  • Cofactor (\(A_{ij}\)): \(A_{ij} = (-1)^{i+j}M_{ij}\).

The determinant of a matrix can be evaluated by expanding along any row or column using cofactors. For a \(3 \times 3\) matrix:

\[ |A| = a_{11}A_{11} + a_{12}A_{12} + a_{13}A_{13} \]

2.4.2 Singular and Non-Singular Matrices

  • Singular Matrix: A square matrix whose determinant is zero (\(|A|=0\)). It does not have an inverse.
  • Non-Singular Matrix: A square matrix whose determinant is non-zero (\(|A| \neq 0\)). It has an inverse.

2.4.3 Properties of Determinants

  • The determinant of a matrix and its transpose are equal (\(|A| = |A^t|\)).
  • Interchanging any two rows or columns changes the sign of the determinant.
  • If two rows or columns are identical, the determinant is zero.
  • If a row or column contains all zeros, the determinant is zero.
  • Multiplying a row or column by a scalar \(k\) multiplies the determinant by \(k\).

2.5 Matrix Inverse

The inverse of a square matrix \(A\), denoted \(A^{-1}\), is a matrix such that \(AA^{-1} = A^{-1}A = I\), where \(I\) is the identity matrix.

2.5.1 Adjoint Method for \(A^{-1}\)

\[ A^{-1} = \frac{1}{|A|} \text{adj}(A) \]

Where \(\text{adj}(A)\) (adjoint of \(A\)) is the transpose of the cofactor matrix.

2.5.2 Row Operations (Elementary Operations)

Elementary row operations are used to transform a matrix into a simpler form (echelon or reduced echelon form) and can also be used to find the inverse of a matrix or solve systems of linear equations.

  • Interchanging two rows (\(R_i \leftrightarrow R_j\)).
  • Multiplying a row by a non-zero scalar \(k\) (\(kR_i\)).
  • Adding a multiple of one row to another row (\(R_i + kR_j\)).

A matrix is in echelon form if it meets specific criteria (leading 1s, zeros below leading 1s, zero rows at bottom). It is in reduced echelon form if, in addition, elements above leading 1s are also zero.

The rank of a matrix is the number of non-zero rows in its echelon (or reduced echelon) form.

2.6 Solving Systems of Linear Equations

Matrices and determinants are powerful tools for solving systems of linear equations.

2.6.1 Homogeneous and Non-Homogeneous Systems

  • Homogeneous System (\(AX=0\)): All constant terms are zero. Always has the trivial solution (\(x=0, y=0, z=0\)). May have non-trivial solutions if \(|A|=0\).
  • Non-Homogeneous System (\(AX=B\), where \(B \neq 0\)): Has at least one non-zero constant term.

2.6.2 Consistency Criteria

For a system \(AX=B\) (with \(n\) unknowns) and its augmented matrix \([A|B]\):

  • Consistent (has solutions): If \(\text{rank}(A) = \text{rank}([A|B])\).
    • Unique Solution: If \(\text{rank}(A) = \text{rank}([A|B]) = n\).
    • Infinite Solutions: If \(\text{rank}(A) = \text{rank}([A|B]) < n\).
  • Inconsistent (no solution): If \(\text{rank}(A) \neq \text{rank}([A|B])\).

2.6.3 Methods for Solving Linear Equations

  • Matrix Inversion Method (\(X=A^{-1}B\)): Used when \(A\) is a non-singular square matrix.
  • Cramer's Rule: Uses determinants to find the values of unknowns. Applicable for systems with a unique solution where \(|A| \neq 0\).
  • Gauss Elimination Method: Uses elementary row operations to transform the augmented matrix into echelon form, then back-substitution.
  • Gauss-Jordan Method: Uses elementary row operations to transform the augmented matrix into reduced echelon form, directly yielding the solution.

Chapter Summary

  • Matrices: Rectangular arrays of numbers (elements) with defined order (\(m \times n\)).
  • Types: Classified by shape (row, column, square, rectangular) or element properties (diagonal, scalar, identity, null, triangular).
  • Algebra: Scalar multiplication, addition/subtraction (same order), and multiplication (compatible dimensions).
  • Determinant (\(|A|\)): A scalar value associated with square matrices, used to find minors and cofactors, and determine singularity.
  • Matrix Inverse (\(A^{-1}\)): Exists only for non-singular matrices (\(|A| \neq 0\)), often found using the adjoint method or row operations.
  • Linear Equations: Matrices solve homogeneous (\(AX=0\)) and non-homogeneous (\(AX=B\)) systems. Consistency and number of solutions are determined by rank. Methods include matrix inversion, Cramer's rule, Gauss elimination, and Gauss-Jordan.
