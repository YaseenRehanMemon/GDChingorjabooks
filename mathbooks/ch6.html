6.1 Fundamental Counting Principles

Before delving into permutations and combinations, it's essential to understand the basic principles of counting, which help determine the number of possible outcomes for a series of events.

6.1.1 Factorial Notation (\(n!\))

The factorial of a non-negative integer \(n\), denoted by \(n!\), is the product of all positive integers less than or equal to \(n\).

\[ n! = n \times (n-1) \times (n-2) \times \dots \times 3 \times 2 \times 1 \]

By definition, \(0! = 1\).

6.1.2 Sum Principle of Counting

If two events are mutually exclusive (cannot happen at the same time), and one can occur in \(m\) ways and the other in \(n\) ways, then the total number of ways for either event to occur is \(m+n\).

6.1.3 Product Principle of Counting

If an event can occur in \(m\) ways, and after it occurs, a second independent event can occur in \(n\) ways, then the total number of ways both events can occur in sequence is \(m \times n\).

6.2 Permutation

A permutation is an arrangement of objects in a specific order. The order of arrangement matters in permutations.

6.2.1 Permutation of \(n\) Different Objects Taken \(r\) at a Time (\(n P_r\))

The number of permutations of \(n\) distinct objects taken \(r\) at a time is given by:

\[ n P_r = \frac{n!}{(n-r)!} \]

6.2.2 Permutation of \(n\) Objects with Repetition

If there are \(n\) objects where \(n_1\) are alike of one kind, \(n_2\) are alike of a second kind, and so on, then the number of distinct permutations of these \(n\) objects is:

\[ P = \frac{n!}{n_1! n_2! \dots n_k!} \]

6.2.3 Circular Permutation

For arrangements in a circle, the starting point is not fixed, so we consider relative positions.

  • If clockwise and anticlockwise arrangements are considered distinct, the number of circular permutations of \(n\) distinct objects is \((n-1)!\).
  • If clockwise and anticlockwise arrangements are considered identical (e.g., a necklace), the number is \(\frac{1}{2}(n-1)!\).

6.3 Combination

A combination is a selection of objects where the order of selection does NOT matter. It is about choosing a subset from a larger set.

6.3.1 Combination of \(n\) Different Objects Taken \(r\) at a Time (\(n C_r\))

The number of combinations of \(n\) distinct objects taken \(r\) at a time is given by:

\[ n C_r = \binom{n}{r} = \frac{n!}{r!(n-r)!} \]

6.3.2 Properties of Combinations

  • \(\binom{n}{r} = \binom{n}{n-r}\) (Complementary combinations are equal).
  • \(\binom{n}{0} = \binom{n}{n} = 1\).
  • \(\binom{n}{r} + \binom{n}{r-1} = \binom{n+1}{r}\) (Pascal's Identity).

6.4 Probability

Probability is a numerical measure of the likelihood that an event will occur. It is expressed as a value between 0 and 1, inclusive.

6.4.1 Key Definitions

  • Statistical Experiment: A process that leads to well-defined outcomes.
  • Sample Space (\(S\)): The set of all possible outcomes of an experiment.
  • Event (\(E\)): Any subset of the sample space.
  • Mutually Exclusive Events: Events that cannot occur at the same time (\(E_1 \cap E_2 = \emptyset\)).
  • Equally Likely Events: Events that have the same chance of occurring.
  • Independent Events: The occurrence of one event does not affect the probability of another.
  • Dependent Events: The occurrence of one event affects the probability of another.
  • Exhaustive Events: Events whose union covers the entire sample space.

6.4.2 Formula for Probability

\[ P(E) = \frac{\text{Number of favorable outcomes for E}}{\text{Total number of possible outcomes in S}} = \frac{n(E)}{n(S)} \]

Properties: \(0 \le P(E) \le 1\).

6.4.3 Conditional Probability

The probability of event \(A\) occurring given that event \(B\) has already occurred is called conditional probability, denoted by \(P(A|B)\).

\[ P(A|B) = \frac{P(A \cap B)}{P(B)} \quad \text{if } P(B) \neq 0 \]

6.4.4 Multiplication and Addition Theorems of Probability

  • Multiplication Rule: \(P(A \cap B) = P(A) \times P(B|A)\). If \(A\) and \(B\) are independent, \(P(A \cap B) = P(A) \times P(B)\).
  • Addition Rule: \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\). If \(A\) and \(B\) are mutually exclusive, \(P(A \cup B) = P(A) + P(B)\).

Chapter Summary

  • Factorial (\(n!\)): Product of integers from 1 to \(n\).
  • Permutation (\(n P_r\)): Ordered arrangement of \(r\) objects from \(n\) distinct objects (\(n P_r = \frac{n!}{(n-r)!}\)). Order matters.
  • Combination (\(n C_r\)): Unordered selection of \(r\) objects from \(n\) distinct objects (\(n C_r = \frac{n!}{r!(n-r)!}\)). Order does not matter.
  • Probability: Measure of likelihood \((P(E) = \frac{n(E)}{n(S)})\). Key concepts include sample space, events, mutually exclusive/independent events.
  • Theorems: Addition rule for \((P(A \cup B))\) and multiplication rule for \((P(A \cap B))\), including conditional probability.
