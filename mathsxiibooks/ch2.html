1.1 Introduction to Functions

A function is a rule that assigns each input value to exactly one output value. We can think of it as a machine that takes an input and produces a unique output.

Domain and Range

  • Domain: The set of all possible input values (often the x-values).
  • Range: The set of all possible output values (often the y-values).

We can identify the domain and range by looking at a function's graph. The domain is the horizontal spread of the graph, and the range is the vertical spread.

Modulus Function

The modulus function, written as \(y = |x|\), gives the absolute value of a number. It essentially makes any negative input positive.

\[ |x| = \begin{cases} x & \text{if } x \ge 0 \\ -x & \text{if } x < 0 \end{cases} \]
  • Domain: All real numbers (\(\mathbb{R}\)).
  • Range: All non-negative real numbers (\(y \ge 0\)).

1.2 Composition of Functions

Composition is an operation where the output of one function becomes the input for another function. It's like linking two function machines together.

Notation and Definition

The composition of function \(g\) with function \(f\) is denoted as \( (g \circ f)(x) \), which is read as "g of f of x".

It is defined as: \( (g \circ f)(x) = g(f(x)) \).

Important: The order matters! \( (g \circ f)(x) \) is generally not the same as \( (f \circ g)(x) \).

Inverse of a Composition

If two functions \(f\) and \(g\) are bijective (one-to-one and onto), the inverse of their composition has a special property:

\[ (g \circ f)^{-1}(x) = (f^{-1} \circ g^{-1})(x) \]

This means the inverse of the composition is the composition of the inverses in the reverse order.

1.3 Types of Functions

Functions can be classified into different categories based on their properties and structure.

Key Function Types

  • Algebraic: Functions built from polynomials using algebraic operations (addition, multiplication, roots). Example: \(f(x) = \sqrt{x^2+1}\).
  • Trigonometric: Functions like \(\sin(x)\), \(\cos(x)\), \(\tan(x)\) that relate angles of a triangle to the lengths of its sides.
  • Exponential: Functions where the variable is in the exponent. Example: \(f(x) = a^x\), where \(a\) is a constant base.
  • Logarithmic: The inverse of exponential functions. Example: \(f(x) = \log_a(x)\).
  • Explicit vs. Implicit:
    • An explicit function is written as \(y = f(x)\), where \(y\) is isolated. Example: \(y = 3x^2 - 5\).
    • An implicit function has \(x\) and \(y\) mixed together. Example: \(x^2 + y^2 = 25\).

1.4 Introduction to Limits

The concept of a limit is fundamental to calculus. It describes the value that a function "approaches" as the input approaches some value.

The Idea of a Limit

We write \(\lim_{x \to a} f(x) = L\), which means: "As \(x\) gets arbitrarily close to \(a\), the value of \(f(x)\) gets arbitrarily close to \(L\)".

The function doesn't actually have to equal \(L\) at \(x=a\). The limit is only concerned with the behavior *near* the point.

Left-Hand and Right-Hand Limits

A limit exists only if the function approaches the same value from both sides.

  • Left-Hand Limit (\(x \to a^-\)): The value the function approaches as \(x\) comes from the left side (values less than \(a\)).
  • Right-Hand Limit (\(x \to a^+\)): The value the function approaches as \(x\) comes from the right side (values greater than \(a\)).
The limit \(\lim_{x \to a} f(x)\) exists if and only if \(\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x)\).

1.5 Evaluating Limits

We can evaluate limits using several theorems and techniques.

Limit Theorems

If \(\lim_{x \to a} f(x) = L\) and \(\lim_{x \to a} g(x) = M\), then:

  • Sum/Difference: \(\lim_{x \to a} [f(x) \pm g(x)] = L \pm M\)
  • Product: \(\lim_{x \to a} [f(x) \cdot g(x)] = L \cdot M\)
  • Quotient: \(\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{L}{M}\) (provided \(M \neq 0\))

Important Trigonometric Limit

A crucial limit used frequently in calculus is:

\[ \lim_{x \to 0} \frac{\sin x}{x} = 1 \]

(Note: \(x\) must be in radians).

1.6 Continuity

A function is continuous if you can draw its graph without lifting your pen from the paper. Formally, it means there are no breaks, jumps, or holes in the graph.

Conditions for Continuity at a Point

A function \(f(x)\) is continuous at a point \(x=a\) if all three of the following conditions are met:

  1. \(f(a)\) is defined (the point exists).
  2. \(\lim_{x \to a} f(x)\) exists (the limit exists).
  3. \(\lim_{x \to a} f(x) = f(a)\) (the limit equals the function value).

Chapter Summary

  • Functions: A rule that maps an input (domain) to a unique output (range).
  • Composition: Creating a new function by applying one function to the result of another, written as \(g(f(x))\).
  • Limits: Describe the value a function approaches as its input approaches a certain point. A limit exists only if the left-hand and right-hand limits are equal.
  • Continuity: A function is continuous at a point if the limit at that point exists, the function value is defined, and both are equal.
