11.1 Functions of Two Variables

While ordinary differentiation deals with functions of a single variable, like \(y = f(x)\), partial differentiation extends these concepts to functions of two or more independent variables, such as \(z = f(x, y)\).

Definition

A function of two variables, \(z = f(x, y)\), assigns a unique output value \(z\) to every ordered pair of input values \((x, y)\) in its domain.

For example, the area of a rectangle is a function of its length and width, \(A(l, w) = l \cdot w\).

11.2 Partial Derivatives

A partial derivative measures the rate of change of a multivariable function with respect to one of its variables, while holding all other variables constant.

Notation and Calculation

For a function \(z = f(x, y)\):

  • The partial derivative with respect to x is denoted \(\frac{\partial z}{\partial x}\) or \(f_x(x,y)\). To calculate it, treat \(y\) as a constant and differentiate \(f\) with respect to \(x\) using standard rules.
  • The partial derivative with respect to y is denoted \(\frac{\partial z}{\partial y}\) or \(f_y(x,y)\). To calculate it, treat \(x\) as a constant and differentiate \(f\) with respect to \(y\).

Example

If \(f(x, y) = x^2y^3 + 2x + y\), find \(f_x\) and \(f_y\).

With respect to x (treat y as constant):

\[ \frac{\partial f}{\partial x} = (2x)y^3 + 2 + 0 = 2xy^3 + 2 \]

With respect to y (treat x as constant):

\[ \frac{\partial f}{\partial y} = x^2(3y^2) + 0 + 1 = 3x^2y^2 + 1 \]

11.3 Homogeneous Functions

A homogeneous function is a special type of multivariable function where scaling all the input variables by a factor \(t\) results in the original function being scaled by some power of \(t\).

Definition

A function \(f(x, y)\) is called a homogeneous function of degree n if for any \(t > 0\):

\[ f(tx, ty) = t^n f(x, y) \]

Example: The function \(f(x, y) = x^2 + 3xy\) is homogeneous of degree 2 because \(f(tx, ty) = (tx)^2 + 3(tx)(ty) = t^2(x^2 + 3xy) = t^2 f(x, y)\).

11.4 Euler's Theorem on Homogeneous Functions

Euler's theorem provides a direct relationship between a homogeneous function and its partial derivatives.

Euler's Theorem

If \(z = f(x, y)\) is a homogeneous function of degree \(n\), then:

\[ x \frac{\partial z}{\partial x} + y \frac{\partial z}{\partial y} = nz \]

This theorem is a powerful tool for verifying if a function is homogeneous and for solving certain types of partial differential equations.
