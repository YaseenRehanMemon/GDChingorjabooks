1.1 What is MAPLE?

Maple is a powerful software for symbolic computation, allowing you to perform complex mathematical operations, visualize data, and solve problems from various areas of mathematics.

Key Features

  • Symbolic Computation: Manipulate mathematical expressions with variables, like solving x + a = 0 for x.
  • Numerical Computation: Calculate with numbers to find approximate solutions.
  • Visualization: Create 2D and 3D plots of functions and data.
  • Programming: Develop custom algorithms and functions.

1.2 The MAPLE Environment

The main Maple interface consists of several key components that help you interact with the software.

Main Components

  • Menu Bar: Provides access to file operations, editing tools, and other features (File, Edit, View, etc.).
  • Toolbar: Contains shortcuts for common actions like creating new files, saving, and printing.
  • Palettes: Collections of clickable templates for mathematical expressions, symbols, and matrices, making it easy to enter complex notation.
  • Workspace: The main area where you type commands and see the output. It can operate in two modes:
    • Document Mode: Allows mixing text and executable math on the same line. Great for creating reports.
    • Worksheet Mode: A more traditional programming style where each command starts with a prompt (>).

1.3 Basic Commands

Commands are the instructions you give to Maple. Every command must end with a semicolon (;) to display the output, or a colon (:) to hide it.

Essential Commands

Command Description
:= Assignment operator. Assigns a value to a variable (e.g., a := 5;).
restart; Clears all variables from memory, providing a fresh start.
evalf(); Evaluates an expression to a floating-point (decimal) number (e.g., evalf(Pi);).
simplify(); Applies simplification rules to an expression.
solve(); Solves an equation or a system of equations for specified variables.

1.4 Working with Polynomials

Maple provides powerful tools for manipulating polynomials.

Polynomial Operations

  • Factoring: Use the factor() command to factor a polynomial.
    Example: factor(x^2 - 4); returns (x - 2)(x + 2).
  • Expanding: Use the expand() command to expand a factored expression.
    Example: expand((x - 2)*(x + 2)); returns x^2 - 4.
  • Substituting: Use the subs() command to substitute a value or expression into another.
    Example: subs(x=3, x^2 + 1); returns 10.

1.5 Graphics and Plotting

Maple's plotting capabilities are one of its most useful features for visualizing functions.

2D Plotting

  • Single Function: plot(f(x), x = a..b); plots the function f(x) from x=a to x=b.
    Example: plot(sin(x), x = 0..2*Pi);
  • Multiple Functions: plot({f(x), g(x)}, x = a..b); plots multiple functions on the same axes.
    Example: plot({sin(x), cos(x)}, x = 0..2*Pi);
  • Parametric Plots: plot([x(t), y(t), t = a..b]); creates a plot from parametric equations.
    Example: plot([cos(t), sin(t), t = 0..2*Pi]); (This draws a circle).

1.6 Matrices

Maple can define and perform operations on matrices and vectors.

Creating and Using Matrices

You can use the Matrix palette or commands. To perform matrix operations, you first need to load the Linear Algebra package:

with(LinearAlgebra):
  • Defining a Matrix: A := Matrix([[1, 2], [3, 4]]);
  • Matrix Operations: You can add (A + B), subtract (A - B), and multiply (A . B) matrices. Note the use of a period for matrix multiplication.
  • Inverse and Transpose: Find the inverse with MatrixInverse(A) and the transpose with Transpose(A).
