12.1 Why Use Numerical Methods?

While analytical methods (like factoring or the quadratic formula) provide exact solutions, many real-world problems are too complex to be solved this way. Numerical methods are iterative algorithms that provide approximate solutions to mathematical problems, which are often sufficient for practical applications.

12.2 Solving Non-Linear Equations (Root-Finding)

Root-finding algorithms are used to find solutions (or roots) for an equation of the form \(f(x) = 0\).

Bisection Method

This method starts with an interval \([a, b]\) where \(f(a)\) and \(f(b)\) have opposite signs, guaranteeing a root exists between them. The interval is repeatedly halved, and the half containing the root is selected for the next iteration.

Regula Falsi Method (False Position)

Similar to the bisection method, Regula Falsi also starts with an interval \([a, b]\) bracketing a root. Instead of finding the midpoint, it finds the x-intercept of the line connecting \((a, f(a))\) and \((b, f(b))\), which often converges to the root faster.

Regula Falsi Formula

The next approximation, \(x_1\), is given by:

\[ x_1 = \frac{a \cdot f(b) - b \cdot f(a)}{f(b) - f(a)} \]

Newton-Raphson Method

This is a powerful and fast method that uses the tangent line to the function to find successively better approximations for a root. It requires both the function \(f(x)\) and its derivative \(f'(x)\).

Newton-Raphson Formula

Starting with an initial guess \(x_0\), the next approximation is:

\[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \]

12.3 Numerical Quadrature (Integration)

Numerical quadrature is the process of finding an approximate value for a definite integral \(\int_{a}^{b} f(x) \,dx\). This is often used when the antiderivative of \(f(x)\) is difficult or impossible to find.

Trapezoidal Rule

The trapezoidal rule approximates the area under a curve by dividing it into a series of trapezoids and summing their areas. For \(n\) subintervals of equal width \(h = \frac{b-a}{n}\):

Trapezoidal Rule Formula

\[ \int_{a}^{b} f(x) \,dx \approx \frac{h}{2} [y_0 + 2y_1 + 2y_2 + \dots + 2y_{n-1} + y_n] \]

Simpson's Rules

Simpson's rules are more accurate methods that use quadratic or cubic approximations for the curve instead of straight lines.

Simpson's 1/3 Rule

This rule approximates the curve using parabolas. It requires an even number of subintervals (\(n\)).

\[ \int_{a}^{b} f(x) \,dx \approx \frac{h}{3} [y_0 + 4y_1 + 2y_2 + 4y_3 + \dots + 2y_{n-2} + 4y_{n-1} + y_n] \]

Simpson's 3/8 Rule

This rule uses cubic polynomials and requires the number of subintervals (\(n\)) to be a multiple of 3.

\[ \int_{a}^{b} f(x) \,dx \approx \frac{3h}{8} [y_0 + 3y_1 + 3y_2 + 2y_3 + 3y_4 + \dots + 3y_{n-1} + y_n] \]
