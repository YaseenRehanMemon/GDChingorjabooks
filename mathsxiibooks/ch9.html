9.1 Defining Conics with Eccentricity

A conic section can be defined as the set of all points P in a plane such that the ratio of the distance from P to a fixed point (the focus) and the distance from P to a fixed line (the directrix) is a constant. This constant ratio is called the eccentricity, denoted by \(e\).

Types of Conics based on Eccentricity (e)

  • If \(e = 1\), the conic is a Parabola.
  • If \(0 < e < 1\), the conic is an Ellipse.
  • If \(e > 1\), the conic is a Hyperbola.

9.2 The Parabola (\(e=1\))

A parabola is the set of all points in a plane that are equidistant from a fixed point (focus) and a fixed line (directrix).

Standard Equations of a Parabola

With vertex at the origin \((0,0)\):

  • Opens right: \(y^2 = 4ax\) (Focus: \((a,0)\), Directrix: \(x=-a\))
  • Opens left: \(y^2 = -4ax\) (Focus: \((-a,0)\), Directrix: \(x=a\))
  • Opens up: \(x^2 = 4ay\) (Focus: \((0,a)\), Directrix: \(y=-a\))
  • Opens down: \(x^2 = -4ay\) (Focus: \((0,-a)\), Directrix: \(y=a\))

9.3 The Ellipse (\(0 < e < 1\))

An ellipse is the set of all points in a plane, the sum of whose distances from two fixed points (the foci) is constant.

Standard Equation of an Ellipse

With center at the origin \((0,0)\) and \(a>b>0\):

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \]
  • Vertices: \((\pm a, 0)\)
  • Co-vertices: \((0, \pm b)\)
  • Foci: \((\pm c, 0)\), where \(c^2 = a^2 - b^2\)
  • Eccentricity: \(e = \frac{c}{a} < 1\)

If the major axis is vertical, the roles of \(a\) and \(b\) are swapped.

9.4 The Hyperbola (\(e>1\))

A hyperbola is the set of all points in a plane, the difference of whose distances from two fixed points (the foci) is a positive constant.

Standard Equation of a Hyperbola

With center at the origin \((0,0)\):

\[ \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 \]
  • Vertices: \((\pm a, 0)\)
  • Foci: \((\pm c, 0)\), where \(c^2 = a^2 + b^2\)
  • Asymptotes: \(y = \pm \frac{b}{a}x\)
  • Eccentricity: \(e = \frac{c}{a} > 1\)

If the transverse axis is vertical, the equation becomes \(\frac{y^2}{a^2} - \frac{x^2}{b^2} = 1\).

9.5 General Equation of Conics

The equation of any conic section can be written in the general form:

\[ Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0 \]

The type of conic can be determined by the discriminant, \(B^2 - 4AC\).

  • If \(B^2 - 4AC < 0\), it's an ellipse (or circle).
  • If \(B^2 - 4AC = 0\), it's a parabola.
  • If \(B^2 - 4AC > 0\), it's a hyperbola.
