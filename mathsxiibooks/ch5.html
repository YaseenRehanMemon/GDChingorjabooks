5.1 Vector Functions

A vector function is a function that takes a real number (often time, \(t\)) as input and produces a vector as output. It can be used to describe the position of a moving particle in space.

Definition and Components

A vector function \(\mathbf{r}(t)\) in three dimensions is defined by its component functions:

\[ \mathbf{r}(t) = f(t)\mathbf{i} + g(t)\mathbf{j} + h(t)\mathbf{k} \]
  • The domain of \(\mathbf{r}(t)\) is the intersection of the domains of its component functions \(f(t), g(t)\), and \(h(t)\).
  • The range is the set of all output vectors.

5.2 Limit and Continuity of Vector Functions

The concepts of limits and continuity extend naturally from scalar functions to vector functions. We simply apply the concepts to each component individually.

Limit of a Vector Function

The limit of \(\mathbf{r}(t)\) as \(t\) approaches \(t_0\) is found by taking the limit of each component:

\[ \lim_{t \to t_0} \mathbf{r}(t) = \left(\lim_{t \to t_0} f(t)\right)\mathbf{i} + \left(\lim_{t \to t_0} g(t)\right)\mathbf{j} + \left(\lim_{t \to t_0} h(t)\right)\mathbf{k} \]

Continuity

A vector function \(\mathbf{r}(t)\) is continuous at \(t = t_0\) if:

\[ \lim_{t \to t_0} \mathbf{r}(t) = \mathbf{r}(t_0) \]

This is true if and only if all of its component functions are continuous at \(t = t_0\).

5.3 Derivative of a Vector Function

The derivative of a vector function, \(\mathbf{r}'(t)\), represents a tangent vector to the curve traced by \(\mathbf{r}(t)\). It gives both the direction and the magnitude of the rate of change.

Calculating the Derivative

The derivative is found by differentiating each component function with respect to the parameter \(t\):

\[ \mathbf{r}'(t) = \frac{d\mathbf{r}}{dt} = f'(t)\mathbf{i} + g'(t)\mathbf{j} + h'(t)\mathbf{k} \]

5.4 Differentiation Rules for Vectors

The standard differentiation rules (like the product rule) have vector equivalents for operations like the dot product and cross product.

Key Vector Differentiation Rules

Let \(\mathbf{u}(t)\) and \(\mathbf{v}(t)\) be vector functions, \(c\) be a scalar, and \(f(t)\) be a scalar function.
  • Sum Rule: \(\frac{d}{dt}[\mathbf{u}(t) + \mathbf{v}(t)] = \mathbf{u}'(t) + \mathbf{v}'(t)\)
  • Scalar Multiple: \(\frac{d}{dt}[f(t)\mathbf{u}(t)] = f'(t)\mathbf{u}(t) + f(t)\mathbf{u}'(t)\)
  • Dot Product Rule: \(\frac{d}{dt}[\mathbf{u}(t) \cdot \mathbf{v}(t)] = \mathbf{u}'(t) \cdot \mathbf{v}(t) + \mathbf{u}(t) \cdot \mathbf{v}'(t)\)
  • Cross Product Rule: \(\frac{d}{dt}[\mathbf{u}(t) \times \mathbf{v}(t)] = \mathbf{u}'(t) \times \mathbf{v}(t) + \mathbf{u}(t) \times \mathbf{v}'(t)\)

5.5 Velocity and Acceleration

In physics, vector derivatives are used to describe motion along a curve.

Motion in Space

If \(\mathbf{r}(t)\) is the position vector of a particle at time \(t\), then:

  • The velocity vector is the first derivative: \(\mathbf{v}(t) = \mathbf{r}'(t)\). It is tangent to the path of motion.
  • The speed is the magnitude of the velocity: \(\|\mathbf{v}(t)\|\).
  • The acceleration vector is the second derivative: \(\mathbf{a}(t) = \mathbf{v}'(t) = \mathbf{r}''(t)\).
