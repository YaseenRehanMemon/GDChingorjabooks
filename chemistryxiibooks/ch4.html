1. The IUPAC System

With millions of organic compounds, a systematic method for naming them is essential. The International Union of Pure and Applied Chemistry (IUPAC) system provides a set of rules that gives each compound a unique and unambiguous name based on its structure.

Core IUPAC Rules

  1. Identify the Parent Chain: Find the longest continuous carbon chain. If a functional group is present, the chain must contain it.
  2. Number the Chain: Number the carbons in the parent chain, starting from the end that gives any substituents or functional groups the lowest possible numbers.
  3. Name the Substituents: Name any branches or groups attached to the parent chain as prefixes (e.g., methyl, chloro, hydroxy).
  4. Assemble the Name: Combine the parts in the order: (substituent number)-(substituent name)(parent chain name). Use prefixes like di-, tri- for multiple identical substituents.

2. Nomenclature of Alkanes

Alkanes are saturated hydrocarbons with the general formula \(C_nH_{2n+2}\). Their IUPAC names end in -ane.

  • Parent Chain: The longest continuous carbon chain.
  • Numbering: Start from the end that gives the substituents the lowest numbers.
  • Example: A 5-carbon chain with a methyl group on the 2nd carbon is named 2-methylpentane.

3. Nomenclature of Alkenes and Alkynes

Alkenes (double bonds) and alkynes (triple bonds) are unsaturated hydrocarbons.

Naming Unsaturated Hydrocarbons

  • Parent Chain: Must include the double or triple bond.
  • Suffix: The ending -ane is changed to -ene for alkenes and -yne for alkynes.
  • Numbering: The chain is numbered to give the first carbon of the multiple bond the lowest possible number. This number is placed before the suffix (e.g., pent-2-ene).
  • Example (Alkene): \( \text{CH}_3\text{CH}=\text{CHCH}_2\text{CH}_3 \) is named pent-2-ene.
  • Example (Alkyne): \( \text{HC}\equiv\text{CCH}_2\text{CH}_3 \) is named but-1-yne.

4. Naming Compounds with Functional Groups

The presence of a functional group changes the suffix of the parent name.

Family Functional Group Suffix Example (and Name)

5. Priority of Functional Groups

When a molecule contains more than one functional group, a priority system determines which group is named with the suffix (the principal group) and which are named as prefixes.

Order of Priority (Highest to Lowest)

Carboxylic Acid > Ester > Acyl Halide > Amide > Aldehyde > Ketone > Alcohol > Amine > Alkene > Alkyne > Alkane > Haloalkane

Example: In a molecule with both an -OH and a -COOH group, it is named as a carboxylic acid, and the -OH group is named with the prefix hydroxy.

Chapter Summary

  • IUPAC System: A systematic set of rules for naming organic compounds based on their structure.
  • Core Structure: A name consists of a prefix (substituents), a parent name (longest carbon chain), and a suffix (principal functional group).
  • Alkanes, Alkenes, Alkynes: Named with suffixes -ane, -ene, and -yne respectively. The position of multiple bonds is indicated by a number.
  • Functional Groups: The presence of a functional group determines the family of the compound and is indicated by a characteristic suffix (e.g., -ol for alcohols, -al for aldehydes).
  • Priority Rules: When multiple functional groups are present, a priority list determines which group is named as the suffix.
